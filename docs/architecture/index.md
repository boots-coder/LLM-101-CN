# 模型架构

本章深入解读大语言模型的核心架构，从基础的 Transformer 出发，逐步演进到 GPT、Llama 和 DeepSeek-V3。

## 架构演进路线

```
Transformer (2017) → GPT (2018) → Llama (2023) → DeepSeek-V3 (2024)
     │                   │              │                │
  Encoder-Decoder    Decoder-Only     开源优化        MoE + MLA
```

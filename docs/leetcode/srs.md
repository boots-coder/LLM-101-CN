# 今日复习池

> 间隔重复（Spaced Repetition）的核心：错过的卡明天复习，连续答对的卡间隔翻倍。**每天花 5-10 分钟扫一遍这里，比集中刷题更稳**。

<SrsReview />

## 算法说明

本站采用 **SM-2 简化版**：

| 答题情况 | 评分 | 间隔变化 |
|---------|------|---------|
| 一次答对 | 4 | 翻倍（×ease，ease 默认 2.5） |
| 重试后对 | 3 | 翻倍但 ease 不增 |
| 看了答案 | 2 | 重置为 1 天 |
| 跳过/连错 | 1 | 重置为 1 天 + ease 衰减 0.2 |

到期判断：`due ≤ 今天` 即出现在上方列表。

::: tip 复习节奏建议
- **每天打开**：先扫今日复习池（5-10 分钟），再去关卡地图学新内容（30-60 分钟）。
- **不要"补做"**：错过一天会损失 streak，但卡片本身的间隔不会因此乱掉，下次答对仍按 SM-2 推进。
- **在地铁/排队时刷**：每张微选择题都很短，碎片时间最适合。
:::
